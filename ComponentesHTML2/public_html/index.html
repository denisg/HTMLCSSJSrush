Frases HTML

EM: se utiliza pra dar significado a una parte del texto

STRONG se utiliza para resaltar elementos

Q se utiliza para poner comillas

CODE se utiliza para poner el texto como si fuera cpdogp de algun lenguaje de programacion

ABBR se utiliza para poner Abreviaciones

ACRONYM se utiliza para poner acrónimos

DFN se utiliza para poner definiciones

KBD significa texto que debe ser introducido por el usuario

SAMP se utiliza para mostrar un ejemplo de una salida de un programa

VAR se utiliza para mostrar que el texto es una variable
