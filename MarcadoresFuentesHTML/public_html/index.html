Marcadores de Fuentes

B se utiliza pra poner en negritas el texto

BIG se utiliza para convertir una fuente mas grande esto es interpretado por el navegador

I se utiliza para convertir el texto en cursiva

S se utiliza para poner una raya en el texto

SMALL se utilzia para jacer el texto mas pequeño

STRIKE es el mismo resultado que S

TT se utiliza para dar formato de teletipo como se utiliza enlos monitores o impresoras antiguas

U se utiliza para subrayar una palabra
